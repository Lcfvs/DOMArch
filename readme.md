DOMArch
=======

A PHP DOM micro-framework, based on [PHPDOM] (https://github.com/Lcfvs/PHPDOM)<br />


Usage :
-------

- Put all files, directly on your /www
- Enable the Apache rewrite module
- Go to [http://127.0.0.1] (http://127.0.0.1)
- Enjoy ;)
